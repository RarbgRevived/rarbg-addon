# rarbg-addon

Stream movies and series from rarbg on Stremio.
Tested with Stremio 3.6.5

# How to run

There is currently no hosted version. Run it yourself:

```
npm start
```

Then start Stremio:


```
./Stremio.sh --services=http://localhost:7000/stremioget/v1
```
